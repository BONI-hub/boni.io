How BNI works?

The magic happens in 3 easy steps

1 Integrate SDK Enable your mobile app to detect BONI spots. As simple as installing a Cordova plugin. Learn more
2 Define Spots Define spot, provide content for it using admin console. Learn more
3 Install Beacons Position BLE beacon (Spot ID) near to the area you want to supplement with virtual context.
