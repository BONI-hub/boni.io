How BNI works?

The magic happens in 3 easy steps

1 Integrate SDK Enable your mobile app to detect BONI Spots. As simple as installing a Cordova plugin.
2 Define Spots Define BONI Spot, provide content for it using admin console.
3 Install Beacons Position Your iBeacon near to the area you want to supplement with virtual context.
