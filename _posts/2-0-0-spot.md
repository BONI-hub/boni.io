BONI Spot is an abstraction of real location (indoor or outdoor) that has corresponding iBeacon. The iBeacon has 3 identifiers:

* UUID
* Major ID
* Minor Is

Each virtual BONI Spot has the same identifiers as well as **Spot name**.
